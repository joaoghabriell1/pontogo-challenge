## :rocket: Rodando o projeto
Para rodar o repositório, siga os seguintes passos::

* Clone o projeto:
```
git clone https://github.com/joaoghabriell1/pontogo-challenge.git
```

* Instale as dependências necessárias:
```
npm install
```
* Para --:
```
npm run codegen
```

* Para rodar o projeto:
```
npm run dev
```
